PRODUCT IMAGES FOLDER
=====================

This folder should contain all product images used in the application.

REQUIREMENTS:
- At least 12 fruit images (apple.jpg, banana.jpg, orange.jpg, etc.)
- At least 12 vegetable images (tomato.jpg, carrot.jpg, lettuce.jpg, etc.)
- Images should be JPG or PNG format
- Recommended size: 200x200 to 400x400 pixels

HOW TO EXPORT IMAGES FROM DATABASE:
1. Compile: javac -d bin src/main/utils/ImageExporter.java
2. Run: java -cp bin main.utils.ImageExporter

HOW TO ADD IMAGES:
1. Place image files in this folder
2. Use Owner dashboard to add products and select images
3. Images are stored in database as BLOBs
